## 操作說明

1. 請先安裝Python
2. 請於cmd（命令提示字元）輸入下列指令安裝套件：
- `pip install Flask`
- `pip install python-dotenv`
- `pip install python-docx`
- `pip install openai`
- `python -m pip install "pymongo[srv]"`
- `pip install openai==0.28`
